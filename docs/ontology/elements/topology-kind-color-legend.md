`src/widgets/topology-map-sigma/ui/SigmaTopology.tsx` renders the topology kind legend that labels the visible node colors for `domain`, `capability`, `element`, and `unknown`.

The legend is the non-color cue paired with the Sigma node fill colors. It keeps the map aligned with WCAG-style guidance that color should not be the only way to convey meaning.
