# Product Owner Operating System Doc

`docs/PRODUCT-OWNER-OPERATING-SYSTEM.md` is the canonical document that defines
Atlas's product-owner gate. It combines problem-first product discovery,
working-backwards customer framing, Shape Up-style slice shaping, RICE-style
priority comparison, design craft, and Atlas-specific ontology stewardship.

The document is intentionally operational, not aspirational. It gives agents a
compact PO pass, verdicts, a quality rubric, and a chief-PO level evaluation so
feature work starts from the phenomenon and ends with runtime evidence.
